# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Revault, please report it responsibly.

**Do not open a public issue.** Instead, email us at **revaulthq@gmail.com** or use [GitHub's private vulnerability reporting](https://github.com/revaultapp/revault/security/advisories/new).

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

|---------|-----------|
